# ACK05を、rekordboxの操作デバイスへ

<p class="hero-lede">OverCUEは、XPPen ACK05のダイヤルと10個のキーをrekordboxのCUE操作へ割り当てる常駐アプリです。macOSとWindowsに対応し、rekordbox Freeプランで利用できます。</p>

<div class="feature-grid">
  <section class="feature-card">
    <h3>10キー＋ダイヤル</h3>
    <p>Cue、Hot Cue、Jump、Quantize、波形操作などを手元にまとめます。</p>
  </section>
  <section class="feature-card">
    <h3>4つのグループ</h3>
    <p>Deck 1、Deck 2、EXPORTなどの設定を切り替えて保存できます。</p>
  </section>
  <section class="feature-card">
    <h3>自由なキーマップ</h3>
    <p>rekordboxの割り当て済みショートカットを検索し、ACK05へ設定できます。</p>
  </section>
</div>
