# 将 ACK05 变成 rekordbox 控制器

<p class="hero-lede">OverCUE 是一款常驻应用，可将 XPPen ACK05 的旋钮和十个按键映射到 rekordbox 的 CUE 操作。支持 macOS 和 Windows，并可配合 rekordbox Free 方案使用。</p>

<div class="feature-grid">
  <section class="feature-card">
    <h3>10 个按键＋旋钮</h3>
    <p>将 Cue、Hot Cue、Jump、Quantize 和波形操作集中到一只手上。</p>
  </section>
  <section class="feature-card">
    <h3>四个分组</h3>
    <p>可以切换并保存 Deck 1、Deck 2、EXPORT 等设置。</p>
  </section>
  <section class="feature-card">
    <h3>自由编辑映射</h3>
    <p>搜索 rekordbox 中已分配的快捷键，并直接映射到 ACK05。</p>
  </section>
</div>
