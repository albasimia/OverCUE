# rekordbox keyboard mapping for OverCUE

`OverCUE-Performance.mappings` is a Windows rekordbox keyboard mapping generated from the
current `Export (Preset)` mapping on this PC. It uses ordinary keyboard shortcuts; F13-F22 are
reserved only for the XP-Pen-to-OverCUE bridge.

1. In rekordbox, open Preferences > Keyboard.
2. Import `OverCUE-Performance.mappings` and select it for PERFORMANCE mode.
3. Close Preferences and focus the deck/waveform area before testing ACK05 input.
4. Restart OverCUE after changing the selected rekordbox mapping.

OverCUE reads the selected PERFORMANCE mapping ID from `rekordbox3.settings` and reads EXPORT
from the mapping named `Export (Preset)` when no explicit EXPORT mapping ID is stored.
