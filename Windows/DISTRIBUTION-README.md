# OverCUE for Windows

[日本語](#日本語) | [English](#english) | [简体中文](#简体中文)

## 日本語

OverCUEは、XPPen ACK05のダイヤルとキーをrekordboxのCUE仕込み操作へ変換します。

### 動作環境

- Windows 10 22H2またはWindows 11（x64）
- XPPen ACK05とXPPen Tabletドライバー
- rekordbox 7

### セットアップ

1. OverCUEとrekordboxを終了します。
2. XPPen Tabletの現在の設定をエクスポートしてバックアップします。
3. `Setup/XPPen/README.md`に従い、同梱の`.pcfg`をインポートします。
4. `Setup/rekordbox/README.md`に従い、同梱のキーボードマッピングをインポートします。
5. rekordboxを起動してから`OverCUE.Windows.exe`を実行します。

XPPenの`.pcfg`をインポートすると現在のXPPen設定が置き換わります。必ず先にバックアップしてください。

本アプリはMicrosoft Storeを経由せずGitHub Releasesから直接配布しています。未署名版ではWindows SmartScreenの「発行元不明」警告が表示される場合があります。Windowsのセキュリティ機能全体を無効にしないでください。

画面右上の「表示言語」から日本語、English、简体中文を切り替えられます。

## English

OverCUE turns the XPPen ACK05 dial and keys into cue-preparation controls for rekordbox.

### Requirements

- Windows 10 22H2 or Windows 11 (x64)
- XPPen ACK05 and the XPPen Tablet driver
- rekordbox 7

### Setup

1. Quit OverCUE and rekordbox.
2. Export a backup of the current XPPen Tablet configuration.
3. Follow `Setup/XPPen/README.md` and import the included `.pcfg` file.
4. Follow `Setup/rekordbox/README.md` and import the included keyboard mapping.
5. Start rekordbox, then run `OverCUE.Windows.exe`.

Importing an XPPen `.pcfg` replaces the current XPPen configuration. Always create the backup first.

This app is distributed directly through GitHub Releases rather than Microsoft Store. An unsigned build may show a Windows SmartScreen unknown-publisher warning. Do not disable Windows security features globally.

## 简体中文

OverCUE 可将 XPPen ACK05 的旋钮和按键转换为 rekordbox 的 CUE 准备操作。

### 系统要求

- Windows 10 22H2 或 Windows 11（x64）
- XPPen ACK05 和 XPPen Tablet 驱动程序
- rekordbox 7

### 设置

1. 退出 OverCUE 和 rekordbox。
2. 导出并备份当前的 XPPen Tablet 配置。
3. 按照 `Setup/XPPen/README.md` 导入随附的 `.pcfg` 文件。
4. 按照 `Setup/rekordbox/README.md` 导入随附的键盘映射。
5. 启动 rekordbox，然后运行 `OverCUE.Windows.exe`。

导入 XPPen `.pcfg` 会替换当前的 XPPen 配置，因此请务必先创建备份。

本应用不通过 Microsoft Store，而是通过 GitHub Releases 直接分发。未签名版本可能显示 Windows SmartScreen 的未知发布者警告。请勿全局关闭 Windows 安全功能。

可在窗口右上角的“显示语言”菜单中切换日本語、English和简体中文。
